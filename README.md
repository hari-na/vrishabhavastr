# Vrishabha Vastr Website

Premium e-commerce website for Vrishabha Vastr - Exquisite Indian Ethnic Wear.

## How to Run

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Deploy to Vercel

1. Push to GitHub
2. Import repository in Vercel
3. Deploy automatically

Built with Vite + Vanilla JS
